== Description ==

Testimonial Gutenberg Block allows users add testimonials and display it anywhere in page/post. With more options and style support you easy to create testimonial as you want with some mouse click. The testimonial is 100% responsive for all devices.

[Live Demo](http://plugins.jmsthemes.com/testimonials/) - [Live Document](http://wp-docs.jmsthemes.com/testimonials/)

You can create a new testimonials in one minute.
- Author Name, Rating, Image, Comment, Position Support
- Unlimited testimonial Slide
- 3 Testimonial Style Support
- Advanced Navigation with 10+ Styles
- Paginagion and Pagination as Author Images
- Easy to change anything : Font, Font size, color, …
- RTL Support
- Multi Language Ready
- More & More …

== Installation ==

1. Install the plugin from within the Dashboard or upload the directory jms-testimonialblock and all its contents to the /wp-content/plugins/ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.

= Minimum Requirements =

* PHP 7.0 or greater is recommended
* MySQL 5.6 or greater is recommended
